Quick start
Create a job, poll for completion, then download the result. The same pattern applies to TTS, images, and videos.

Text-to-Speech
curl -X POST "https://69labs.vip/api/v1/tts/generate" \
  -H "Authorization: Bearer vk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello from the API.",
    "voiceId": "21m00Tcm4TlvDq8ikWAM",
    "modelId": "eleven_multilingual_v2"
  }'
curl "https://69labs.vip/api/v1/tts/status/job_123" \
  -H "Authorization: Bearer vk_your_api_key"
curl -L "https://69labs.vip/api/v1/tts/download/job_123" \
  -H "Authorization: Bearer vk_your_api_key" \
  -o job_123.mp3
Image Generation
curl -X POST "https://69labs.vip/api/v1/images/generate" \
  -H "Authorization: Bearer vk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "A futuristic city skyline at sunset"}'
curl "https://69labs.vip/api/v1/images/status/cm7abc123def" \
  -H "Authorization: Bearer vk_your_api_key"
curl -L "https://69labs.vip/api/v1/images/download/cm7abc123def" \
  -H "Authorization: Bearer vk_your_api_key" \
  -o image.png
Video Generation
curl -X POST "https://69labs.vip/api/v1/videos/generate" \
  -H "Authorization: Bearer vk_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A drone shot over a tropical beach",
    "model": "grok-imagine-video",
    "aspectRatio": "16:9",
    "duration": "10"
  }'
curl "https://69labs.vip/api/v1/videos/status/cm7xyz789ghi" \
  -H "Authorization: Bearer vk_your_api_key"
curl -L "https://69labs.vip/api/v1/videos/download/cm7xyz789ghi" \
  -H "Authorization: Bearer vk_your_api_key" \
  -o video.mp4

GET
/api/v1/usage
Pay-per-use ledger summary for the authenticated API key owner.

curl "https://69labs.vip/api/v1/usage?startDate=2026-05-01&endDate=2026-05-31" \
  -H "Authorization: Bearer vk_your_api_key"
Omit both dates for the current UTC month through today. Custom ranges are inclusive UTC dates and cannot span more than 366 days.
Text-to-Speech
Generate audio first, then request SRT subtitles from the completed TTS job and poll the SRT status before downloading the subtitle file.

POST
/api/v1/tts/generate
Create a TTS job. Returns a jobId and queue position.

Request body
text
Required. 1-500000 characters.
voiceId
Required. Voice ID for the selected voiceProvider; use the full catalog voice_id for catalog voices.
voiceName
Optional. Display name for the voice.
voiceProvider
Optional wire-level value: elevenlabs | edgetts | minimax. edgetts uses discounted finite-tier TTS credits. Default: elevenlabs.
modelId
Optional. Default: eleven_multilingual_v2. Catalog voice mode defaults to speech-02-hd and accepts speech-2.8-hd / speech-2.8-turbo. edgetts uses edge-tts.
voiceSettings
Optional. stability (0-1), similarityBoost (0-1), style (0-1), speed (0.7-1.2), useSpeakerBoost.
minimaxSettings
Optional for catalog voice mode. speed (0.01-10), pitch (-12 to 12), volume (0.5-2), languageBoost (default auto; ISO hints normalize server-side).
Response
{
  "id": "job_123",
  "status": "PENDING",
  "queuePosition": 3
}
GET
/api/v1/tts/status/:jobId
Check job status, chunk progress, and censored retry details when applicable.

Status values
PENDING
Queued and waiting for capacity.
PROCESSING
Actively being generated.
FINALIZING
Output is downloading to storage.
COMPLETED
Ready for download.
FAILED
Generation failed. Credits are refunded.
CANCELLED
Cancelled before processing.
CENSORED
TTS only. Rewrite blocked chunks and submit /tts/retry-censored.
Response
{
  "id": "job_123",
  "status": "CENSORED",
  "createdAt": "2026-01-02T10:00:00.000Z",
  "startedAt": "2026-01-02T10:01:00.000Z",
  "completedAt": null,
  "chunksTotal": 4,
  "chunksCompleted": 2,
  "blockedChunks": [
    { "index": 2, "text": "The original flagged chunk text" }
  ],
  "retryExpiresAt": "2026-03-25T00:00:05.000Z"
}
POST
/api/v1/tts/retry-censored/:jobId
Submit corrected text for censored chunks and resume processing.

Request body
{
  "correctedChunks": [
    {
      "index": 2,
      "text": "Rewritten text that avoids the content moderation block"
    }
  ]
}
Response
{
  "message": "Retry submitted",
  "jobId": "job_123"
}
Corrected text is capped at 5000 characters per chunk. Chunked jobs have a 6-hour retry window.
GET
/api/v1/tts/download/:jobId
Download the audio output file.

curl -L "https://69labs.vip/api/v1/tts/download/job_123" \
  -H "Authorization: Bearer vk_your_api_key" \
  -o job_123.mp3
Files are retained until the account retention window expires. Expired outputs return 410 GONE.
POST
/api/v1/tts/generate-srt/:jobId
Queue SRT subtitle generation for a completed TTS job.

Request
curl -X POST "https://69labs.vip/api/v1/tts/generate-srt/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer vk_your_api_key"
The TTS job must be COMPLETED and its audio file must still be available. Credit-based accounts need subtitle access on the active tier.
Response
{
  "jobId": "123e4567-e89b-12d3-a456-426614174000",
  "srtStatus": "PENDING",
  "creditCost": 0
}
Credit-based SRT requests are free. Pay-per-use responses include billing metadata when SRT PPU is enabled.
GET
/api/v1/tts/srt-status/:jobId
Check SRT generation progress and download availability.

SRT status values
null
SRT has not been requested for this job.
PENDING
SRT generation is queued.
PROCESSING
Audio is being transcribed into subtitles.
COMPLETED
SRT file is ready to download.
FAILED
SRT generation failed. Credits or PPU reservations are released.
Response
{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "srtStatus": "COMPLETED",
  "srtError": null,
  "srtFileAvailable": true,
  "creditCost": 150
}
If SRT has never been requested, srtStatus and creditCost are null.
GET
/api/v1/tts/download-srt/:jobId
Download the generated SRT subtitle file.

curl -L "https://69labs.vip/api/v1/tts/download-srt/123e4567-e89b-12d3-a456-426614174000" \
  -H "Authorization: Bearer vk_your_api_key" \
  -o subtitles.srt
Returns 400 until SRT generation is completed. Expired or deleted SRT files return 410 GONE.

Tools
Synchronous utility actions that spend tool credits. Image compression returns optimized image bytes directly instead of creating a job.

POST
/api/v1/tools/image-compressor/compress
Compress one JPEG, PNG, WebP, or AVIF image up to 10 MB. Supports mode=lossy or mode=lossless.

curl -X POST "https://69labs.vip/api/v1/tools/image-compressor/compress" \
  -H "Authorization: Bearer vk_your_api_key" \
  -F "mode=lossy" \
  -F "file=@thumbnail.jpg" \
  -o thumbnail-compressed.jpg
Response headers include X-Tool-Cost, X-Original-Size-Bytes, X-Compressed-Size-Bytes, X-Saved-Percent, and X-Used-Original.

Errors and limits
All errors follow a consistent JSON format with error and code fields.

Error codes
UNAUTHORIZED
Missing or invalid API key.
FORBIDDEN
API access not enabled for the account.
BAD_REQUEST
Invalid input or job state.
PAYMENT_REQUIRED
Insufficient monthly credits.
NOT_FOUND
Job not found.
GONE
Output file expired.
CONFLICT
Duplicate or in-progress request state.
TOO_MANY_REQUESTS
Rate limit exceeded.
TTS_CHARACTER_QUOTA_EXCEEDED
Rolling TTS character quota exceeded.
VALIDATION_ERROR
Request validation failed.
Error format
{
  "error": "Human readable message",
  "code": "MACHINE_READABLE_CODE"
}

Suggested flows
TTS generation
POST /api/v1/tts/generate to create a job.
Poll GET /api/v1/tts/status/:jobId every 2-5 seconds.
If status is CENSORED, rewrite blockedChunks and POST /api/v1/tts/retry-censored/:jobId.
When status is COMPLETED, download the audio.
SRT subtitles
Wait for the TTS job to reach COMPLETED.
POST /api/v1/tts/generate-srt/:jobId to queue subtitle generation.
Poll GET /api/v1/tts/srt-status/:jobId every 2-5 seconds.
When srtFileAvailable is true, download with GET /api/v1/tts/download-srt/:jobId.
Image generation
POST /api/v1/images/generate to create a job.
Poll GET /api/v1/images/status/:jobId every 3-5 seconds.
When COMPLETED, download the image.
Video generation
POST /api/v1/videos/generate to create a job.
Poll GET /api/v1/videos/status/:jobId every 5-10 seconds.
When COMPLETED, download the video.
Motion graphics
GET /api/v1/motion-graphics/templates to list Remotion templates currently enabled by admins.
POST /api/v1/motion-graphics/render with templateId and props to create a render job.
Poll GET /api/v1/motion-graphics/status/:jobId every 5-10 seconds.
When COMPLETED, download the MP4 output.
Image compression tool
POST multipart /api/v1/tools/image-compressor/compress with file or image and mode=lossy|lossless.
Save the binary image response directly to disk.
Read X-Original-Size-Bytes, X-Compressed-Size-Bytes, X-Saved-Percent, and X-Tool-Cost headers for reporting.
Model discovery
GET /api/v1/models to list all available image/video models and motion templates.
Use /api/v1/motion-graphics/templates when you only need motion template metadata.
Check cost, aspect ratios, and capabilities before submitting jobs.
Voice clone TTS
GET /api/v1/voice-clones to list your cloned voices, or /voice-clones/library for global voices.
POST /api/v1/voice-clones/generate with voiceCloneId and text.
Poll GET /api/v1/tts/status/:jobId every 2-5 seconds.
When COMPLETED, download with GET /api/v1/tts/download/:jobId.
Batch generation
Submit multiple jobs and store each jobId.
Poll with exponential backoff to reduce traffic.
Download outputs as they complete.
Expired output
If download returns 410, the file has expired.
Output retention depends on the account policy returned by model discovery or the user tier.
Re-submit the same input to generate a fresh output.
